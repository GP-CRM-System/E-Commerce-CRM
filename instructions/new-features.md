# New Features

1. Multi-tenancy, support for multiple organizations to use the service at one
2. Uploading and parsing of already existing Excel Sheets or CSV files for importing into the CRM system
3. More focus on E-Commerce start-ups and businesses
4. Market Segmentation to identify growing groups that can be targeted with specific marketing strategies
5. Lead scoring based on scraping publicly available data
6. More OAuth Options like logging in with Google, Microsoft, etc…..
7. Account verification to decrease spam using OTPs or emails
8. Audit Logs for logging and monitoring employee performance and scoring
9. More integrations for social media analytics and third party providers for SMS and Mailing
10. Possible payment integration using local services
11. PDF generation for receipts and offers
12. Campaigns using Emails/SMS for better customer retention
13. Automated Ticket handling and Task Assignment for employees based on score and status
