# Imports & Exports

## Imports

Import customers, products, and orders from CSV or XLSX files with customizable field mapping.

### Create Import

```http
POST /imports
```

**Body:**

```json
{
    "entityType": "customers",
    "fileUrl": "/uploads/import.csv",
    "fieldMappings": {
        "email": "Email",
        "firstName": "First Name",
        "lastName": "Last Name",
        "city": "City"
    }
}
```

### Check Import Status

```http
GET /imports/:id
```

**Response:**

```json
{
    "id": "imp_xxx",
    "entityType": "customers",
    "status": "completed",
    "totalRows": 100,
    "processedRows": 100,
    "successfulRows": 98,
    "failedRows": 2,
    "errors": [
        {
            "row": 15,
            "field": "email",
            "message": "Invalid email format"
        }
    ]
}
```

### Get Import Errors

```http
GET /imports/:id/errors
```

## Exports

Export data to CSV or XLSX files.

### Create Export

```http
POST /exports
```

**Body:**

```json
{
    "entityType": "customers",
    "format": "csv",
    "fields": ["email", "firstName", "lastName", "city"],
    "filters": {
        "source": "shopify"
    }
}
```

### Download Export

```http
GET /exports/:id/download
```

Returns the exported file when status is `completed`.

## Supported Formats

- **CSV**: Comma-separated values
- **XLSX**: Excel format

## Entity Types

- `customers`
- `products`
- `orders`
